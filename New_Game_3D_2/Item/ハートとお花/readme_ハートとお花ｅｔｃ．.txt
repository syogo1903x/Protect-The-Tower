用法


・『汗』と『キラッ』は、そのままではカラーチェンジ不可です。

・影無しのモデルと影有りのモデルを同梱してあります。

・AutoLuminous対応。各発光モーフもしくは点灯モーフで光ります。

・透明モーフ類で透過できます。
　透過させる際には、描画順にお気をつけ下さい。

・明度モーフを操作しないと、色相モーフを操作しても色が変わりません。
　色相0で明度モーフを操作すると、モノトーン系の色になります。
　（カラーチェンジの仕組みは同梱の『テクスチャを使った無断階カラーチェンジの仕組み.png』を参照して下さい。）

・ふわふわお花は、プリセットカラー以外の色をご希望の場合、、色初期化モーフを使用してから右上枠の
　カラーチェンジモーフをご利用下さい。

・色関連のモーフは、0以下にしても1以上にしても大丈夫です。
　また、大きさが足りないようでしたら、拡大モーフを1以上にしてご利用下さい。

・ふわふわお花を拡大すると、物理がおかしくなります。（未熟なモデルで申し訳ない）
　大きくして使いたい場合は、物理をOFFにしてご利用下さい。
